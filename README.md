# Profil Peserta Webdev Workshop

Repository ini digunakan untuk latihan Git dan GitHub pada workshop web development.

Peserta akan belajar:

- Fork repository
- Clone repository hasil fork
- Menambahkan foto profil
- Membuat halaman profil HTML
- Mengedit halaman utama
- Commit perubahan
- Push ke repository fork
- Membuat Pull Request
